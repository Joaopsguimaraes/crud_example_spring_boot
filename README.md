## 🏁 Getting Started <a name = "getting_started"></a>

Inicie o container Postgres para realizar o teste da aplicação:

```shell
docker compose up -d
```

### Prerequisites
1. Java 23 instalado.
2. Gradle 8.3
3. Docker
